# Nova Cinema Suite

Переосмысленная версия проекта бронирования билетов в кинотеатр. Приложение разделено на публичную витрину, подбор мест, оформление билета и административную панель. Вся клиентская логика переписана, классы и структура файлов изменены, стили обновлены.

## Структура

- `public/`
  - `home.html` — афиша и расписание
  - `auditorium.html` — выбор мест
  - `checkout.html` — подтверждение и QR-код
  - `admin.html` — панель управления
  - `auth.html` — вход администратора
- `assets/css/main.css` — единый стиль
- `assets/js/`
  - `core/` — API-клиент, календарь, состояние
  - `customer/` — сценарии публичной части
  - `admin/` — управление кинотеатром
  - `vendor/QRCreator.js` — библиотека генерации QR
  - `media/` — изображения

## Запуск

1. Откройте нужную страницу из каталога `public`.
2. Для отображения данных требуется сеть: все запросы идут на `https://shfe-diplom.neto-server.ru`.
3. Данные для входа администратора:
   - Логин: `shfe-diplom@netology.ru`
   - Пароль: `shfe-diplom`

## Особенности

- Новые названия классов и компонентов интерфейса.
- Модульный JavaScript с переиспользуемым API-клиентом и календарём.
- Проверка авторизации в админке и раздельные сценарии для каждого раздела.
- Валидация выбранных мест перед бронированием и безопасная отрисовка контента.
